# Neithal Prince — Developer Portfolio

🌐 **Live Portfolio**: [https://neithal.vercel.app/]

A modern, fully animated, neon-themed personal portfolio website built using HTML, CSS, Bootstrap, and AOS.
Designed to showcase skills, projects, experience, achievements, and contact details with a premium futuristic UI.

## ⚡ Overview

This portfolio highlights:

- **Skills**: Full-Stack Development & Machine Learning expertise
- **Projects**: Featured works including YOLOv8 ML models and MERN stack applications
- **Experience**: Software Development Engineering Intern at Algon Solutions Pvt. Ltd.
- **Education**: B.Tech in Computer Science & Engineering (UKF College of Engineering & Technology, expected 2026, CGPA: 7.02)
- **Achievements**: 1st Prize Winner in TinkerHub HerHack Hackathon 3.0 (2024)

The UI features a sleek, modern landing hero section with gradient-glow effects, fully responsive interface optimized for all devices, and focuses on Glassmorphism, Neon gradients, smooth animations, and micro-interactions for a clean, modern feel.

## ✨ Features

### 🎨 Dark Neon UI
- Animated radial mesh backgrounds with floating particles
- Gradient text effects and glowing elements
- Glassmorphism cards with subtle transparency
- Custom neon color scheme (cyan, magenta, purple, yellow)

### 🎭 Animations
- Smooth scroll animations powered by AOS (Animate on Scroll)
- Floating particle effects
- Interactive hover effects on buttons, cards, and skills
- Gradient text animations and glowing borders

### 🛠️ Sections Included

- **Hero Section**: Introduction with animated status badge, gradient title, call-to-action buttons, and key stats (CGPA: 7.02, 3+ Projects, 1st Prize)
- **About Me**: Personal background, education details, and achievement highlights
- **Skills**: Categorized technical skills in Frontend, Backend, Machine Learning, and Tools
- **Featured Projects**: Showcase of 3 key projects with descriptions and tech stacks
- **Experience Timeline**: Professional work experience with detailed highlights
- **Contact Form**: Contact information, social links, and downloadable resume

### 📄 Extras
- Downloadable Resume button (PDF)
- Smooth scroll-to-section navigation
- Mobile-friendly responsive layout with custom breakpoints
- Custom scrollbar styling
- Social media integration (GitHub, LinkedIn, Email)

## 🛠️ Tech Stack

### Frontend
- **HTML5**: Semantic markup and structure
- **CSS3**: Custom properties (CSS Variables), Flexbox, Grid, Animations
- **JavaScript**: Interactive elements and DOM manipulation

### Styling & UI
- **Bootstrap 5**: Responsive grid system and components
- **Google Fonts**: Space Grotesk and JetBrains Mono for typography
- **Font Awesome**: Icon library for visual elements

### Animations & Effects
- **AOS (Animate on Scroll)**: Scroll-triggered animations
- **CSS Animations**: Keyframe animations for particles, gradients, and hover effects

### Design Patterns
- **Glassmorphism**: Translucent elements with backdrop blur
- **Neon Gradients**: Vibrant color schemes with glow effects
- **Responsive Design**: Mobile-first approach with custom media queries

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Neithuz-dev/portfolio.git
   cd portfolio
   ```

2. **Open in browser**:
   - Simply open `index.html` in any modern web browser
   - No build process or dependencies required

3. **Customization**:
   - Edit `index.html` to update content
   - Modify `style.css` for styling changes
   - Replace images in `assests/` folder

## 📱 Responsive Design

The portfolio is fully responsive with custom breakpoints:
- **Desktop**: 1024px and above
- **Tablet**: 768px to 1023px
- **Mobile**: 480px to 767px
- **Small Mobile**: Below 480px
